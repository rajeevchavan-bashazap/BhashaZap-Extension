BhashaZap Extension Test Page

How to Test:

  1. Make sure BhashaZap extension is loaded and enabled
  2. Click on the extension icon to select Indian languages (maximum 2)
  3. Double-click on any English word below to see translations
  4. The translation popup should appear near your cursor

Troubleshooting:

  • If translations don't appear, check that the extension is enabled
  • Make sure you've selected at least one Indian language in the extension popup
  • Try refreshing the page after enabling the extension
  • Check browser console (F12) for any error messages
